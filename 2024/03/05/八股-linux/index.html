2024-03-05 八股

- 你都接触过哪些linux命令？

  • 较常用的就是一些目录切换的命令比如cd..，cd ~，cd /
  • 还有一些目录操作命令比如，ls，ll，mkdir，pwd等
  • 还有就是一些文件操作命令，比如touch，cgat，vim等（引申到平时工作使用的tail -n filename查看日志），还有就是修改文件的权限chmod（一般都是直接chmod filename 777），还有就是结合别的命令使用的grep(引申到在日志中查找关键词附近的日志cat -n filename | grep —C 5 "关键字"，其中-C 5表示匹配字符串的前后5行)
  • 还有一些查看系统状态的命令，如top（实时查看系统的cpu使用率，内存使用率，进程信息等），free（查看系统的内存使用情况），ps（于查看系统中的进程信息，包括进程的 ID、状态、资源使用情况等），ps -aux查看当前系统运行的进程
  • 还有一些网络通信的命令，比如ping，ifconfig（查看系统的网络接口信息，包括网络接口的 IP 地址、MAC 地址、状态等），netstat（可以查看当前的网络连接情况、监听端口、网络协议等）
  • 还有一些常用的比如，sudo+其它命令，kill -9 pid（杀死进程）

- top和tail命令是什么意思？

top就是查看正在运行的进程的性能信息

tail就是查看文件的内容，不断刷新文件的最后内容

- netstat什么意思？

显示当前的网络情况，比如网卡设备，统计信息啥的

- NAT是什么？

是用于在本地网络中使用私有地址，在连接互联网时转而使用全局 IP 地址的技术

- 知道pid怎么查这个进程

ps aux | grep pid 其中pid是要查的进程的pid

- 怎么查看内存使用情况？

free命令

- lsof怎么用？如何查看端口被哪个进程占用？

lsof ［options］ filename 显示打开指定文件的所有进程

lsof -i:端口号 查看某一端口的占用情况

- 查看磁盘占用情况，哪个目录用的磁盘空间最多？

df -h查看磁盘占用情况

du -s /* | sort -nr 看那个目录占用空间大

然后再用du -s /root/* | sort -nr一层一层排查

  • ps什么意思？（如何根据进程名查找进程，通过进程ID查找进程？）

    查看正在运行的进程， ps aux查看所有的进程

ps -ef | grep "进程名称"

ps -p <PID>

- linux常见的文件类型有哪些？

  • 普通文件：文本文件或者二进制文件
  • 目录文件：文件夹
  • 链接文件：类似于windows中的快捷方式，包括硬链接、软链接等等
  • 套接字文件：socket
  • 字符设备文件
  • 设备块文件
  • 管道文件：命名管道将文件保留在文件系统中，它也称为FIFO，也就是first in first out。虽然命名管道文件保留在文件系统中，但是这个文件只是使用命名管道的一个入口，在使用命名管道传输数据的时候，仍然是在内存中进行的。

- Linux常用查看日志的命令有哪些？

  1. 查看实时变化的日志：tail -n filename（可以指定显示行数，默认是最后10行）
  2. 搜索关键字附近的日志：cat -n filename | grep "关键字"
  3. 直接进入vim模式，输入/关键字，然后按enter查找，查找下一个就按n

- find命令如何使用？（查看某个文件在哪）

find /path/to/search -name "filename"

- 端口号查进程号

lsof -i :端口号
